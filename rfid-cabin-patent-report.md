# 座舱RFID物品智能管理系统 - 专利检索报告

## 技术方案概述

**一句话描述：** 利用车内RFID天线阵列+UWB融合定位，实时追踪随身物品位置，下车前主动提醒遗漏

**完整框架（A-H）：**
```
由于[驾驶员/乘客随身携带多件物品（手机、钱包、钥匙、眼镜等）]
发生了[现有方案无物品追踪，下车后发现物品遗落在车内，需返回取物]
系统通过[物品RFID标签识别 + 车内三维定位 + 座位关联 + 下车意图检测]
调用了[车内RFID/UWB融合天线阵列（4-6个）、座椅压力传感器、
       车门传感器、车机屏幕、手机APP通知]
最终解决了[车内物品遗落无法及时发现和提醒]的问题
实现了[物品定位精度±20cm，遗漏提醒准确率98%，用户返回取物次数减少85%]
相比现有技术[消费级AirTag无车机联动，无下车意图预测]
技术壁垒在于[车内复杂金属环境下的RFID信号干扰消除 + 物品-座位-人员关联算法]
```

### 创造性评估

**非显而易见性论证：**

1. **技术组合非显而易见**
   - RFID用于物品识别（已知）
   - UWB用于精确定位（已知）
   - 但**车内复杂金属环境下的RFID+UWB融合**存在技术障碍
   - 需要解决信号反射、多径干扰、金属屏蔽等问题

2. **应用场景迁移非显而易见**
   - 消费级AirTag用于户外（已知）
   - 但**车内专用天线阵列部署**是新的技术挑战
   - 需要考虑座椅金属框架、车窗、仪表盘的信号影响

3. **功能组合非显而易见**
   - RFID定位（已知）
   - 下车检测（已知）
   - 但**"定位+意图预测+主动提醒"的闭环**是新的功能组合
   - 需要解决误报问题（何时提醒、如何提醒）

**结论：** ✅ **具备创造性**

## 差异化策略建议

### 权利要求布局

**独立权利要求（最大保护范围）：**
```
一种车内物品智能管理系统，其特征在于：
- 车内部署RFID/UWB融合天线阵列
- 物品贴附RFID标签
- 系统实时追踪物品位置
- 检测下车意图时主动提醒遗漏物品
```

**从属权利要求（逐步限定）：**
1. 天线阵列部署位置（座椅下、门框、顶棚）
2. UWB与RFID的融合定位算法
3. 下车意图检测方法（车门+座椅压力+钥匙距离）
4. 提醒方式（车机屏幕+手机APP+语音）
5. 物品-座位-人员关联算法
